# Haeundae Weather Tracker

Premium web dashboard for monitoring real-time weather in Haeundae-gu, Busan.

## Key Features
- **Real-time Weather**: Data collection via Open-Meteo API.
- - **24-hour Forecast**: Hourly temperature and weather status (drag scroll supported).
  - - **Modern Design**: Glassmorphism and dynamic background themes.
    - - **Responsive UI**: Optimized for all devices.
     
      - ## Tech Stack
      - - HTML5 / CSS3 (Vanilla)
        - - JavaScript (Vanilla)
          - - API: Open-Meteo
            - - Icons: Lucide Icons
              - - Fonts: Google Fonts (Outfit)
               
                - ## Installation
                - 1. Clone this repository.
                  2. 2. Open `index.html` in your browser.
